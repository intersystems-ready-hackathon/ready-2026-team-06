WWWAITEST




1. Agent should ask for a description of a new application in the users language include an email to receive the link for the new app 
e.a. user answers with: Room properties to be used as room masterdata within a hotel software. Room number should be the unique field. Send application link to abc@test.com

2. Agent sends the text to AI to get the language out of the text
using AIGetLanguage  (get Language = EN)

3. Agent sends text to AI to get the email out from the text
using AIGetEmail

4. Agent sends text to AI get a short name for the app
using AIGetAnswer  (Answer = program Name)

5. Agent sends the text to AI to get all possible property names plus other infos form the text
using AIGetProperties  (Answer = csv file of properties)

6. Agent sends the property list to external application to generate the class
using AICreateClass   (Answer = Class name)

7. Get the Source Code of the Class
using GetClass  (Answer = Class as XML)

8. Agent sends the class name to external application to generate the form application out of the class an get the link to the new app
using AICreateForm  (Answer = Link to the App)

9. Open link for use
